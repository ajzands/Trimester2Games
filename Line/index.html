Linetap

This is the first game I created called Linetap, in Linetap, a line moves across the screen. As time ticks down, you must click on the line, for points and more time. The bigger darker blue area gives 1 point, and increases time by 1 second, and the smaller bar in the middle adds 5 points and increases the time by 3 seconds.

Time Worked On: 3 days

With the creation of Linetap, I learned about how to constrain objects or sprites in a game, as well as how to create boundaries for the game itself.

My next challenge will be to try and recreate an old game like Breakout to see if I can create a simple game like it, and possibly add my own aspects to it.
